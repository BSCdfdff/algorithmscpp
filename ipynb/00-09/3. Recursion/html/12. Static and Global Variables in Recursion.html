Static Variable¶

Remember static variable inside a function is like a global variable, but local to function.

That is it maintains its state, like a counter

static int x = 0

In [5]:
int fun1(int n){
    static int x = 0;
    if (n > 0){
        x++;
        return fun1(n-1)+x;
    }
    return 0;
}
